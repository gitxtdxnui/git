关于我们

 

           海南省体育彩票管理中心是经海南省人民政府批准成立的，隶属于海南省文体厅的事业单位，是中国体育彩票在海南省的销售机构，负责中国体育彩票在海南省内的销售、管理、兑奖、服务等工作。

         “海南体彩”中国体育彩票电话销售系统是经国务院财政部授权批准，在海南省体育彩票管理中心委托下研发出的手机购彩客户端软件，借助手机操作系统丰富的多媒体展示功能、便捷的人机交互功能和无线数据传输功能，通过体育彩票电话销售系统实时在线销售电脑体育彩票以供彩民在线购买体育彩票。实现了彩票的无纸化和信息化；财政部和文体厅授权批准，真正保证彩民购彩利益；手机用户通过手机即可随时随地购买彩票，极大的方便了彩民，随时随地，轻松购彩！不受时间地点制约，替代了传统彩票销售受时间、距离的限制，属于方便、低碳、环保的彩票销售方式。
